# GetFinance

## Visão Geral
<!-- AUTO: atualizado pelo /obsidian -->
Pipeline ETL em Python que busca dados financeiros da API Pluggy.ai (open banking brasileiro) e os processa com PySpark. O objetivo é consolidar transações e contas bancárias em um banco de dados local via SQLAlchemy.

## Stack
<!-- AUTO -->
- **Linguagem:** Python
- **Processamento de dados:** PySpark
- **API:** Pluggy.ai (open banking)
- **ORM / Banco:** SQLAlchemy
- **Config:** python-dotenv
