## Automatização de Commit

Foi criado um script Bash para executar automaticamente os comandos:

- `git add .`
    
- `git commit -m "update"`
    
- `git push`
    

---

### Local do arquivo

O script foi salvo em:

```
~/bin/commit
```

---

### Como usar

Basta executar no terminal:

```
commit
```
