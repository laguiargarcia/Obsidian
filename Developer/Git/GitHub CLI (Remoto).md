### Instalação

| Comando | Descrição |
|--------|----------|
| `sudo apt install gh` | Instala o GitHub CLI |

---

### Autenticação

| Comando | Descrição |
|--------|----------|
| `gh auth login` | Faz login na sua conta do GitHub |

---

### Repositórios

| Comando | Descrição |
|--------|----------|
| `gh repo create` | Cria um novo repositório no GitHub |
